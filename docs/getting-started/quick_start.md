## Start Macro

There is already a built in **Coordinate** and **Strategy** but you can still config them yourself.

**Note:** The Default Strategy is only for 6 slots unlocked (**Takaroda** at slot 5 and **Speedwagon** at slot 6)

### Config Team

- Config your team in `Config > Placements` with your matching team

**Note:** Spot Finder Unit is for finding correct spot with correct alignment

### Config Tasks

- Setup your task `Task`
- **Route:** Main Task then do Custom task until timer for Main Task is reset
- Then just start your macro with **F2**
